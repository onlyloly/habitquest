# ⚔️ HabitQuest

A gamified habit tracking application that transforms daily routines into engaging quests.

## ✨ About

HabitQuest helps users build healthy habits through gamification mechanics such as:

* 🎯 Daily quests
* ⚡ Experience (XP) system
* 🔥 Streak tracking
* 🏆 Achievements
* 📊 Progress statistics
* 🌍 Multi-language support
* 🎨 Modern RPG-inspired interface

Instead of simply checking tasks off a list, users progress through levels, earn rewards, and visualize their personal growth.

## 🚀 Features

### 🏠 Home

* User progression system
* XP and level tracking
* Daily quest management
* Achievement preview

### ⚡ Habits

* Create custom habits
* Complete daily quests
* Delete habits
* Category filtering
* Search functionality

### 📊 Statistics

* Weekly XP analytics
* Monthly progress charts
* Category breakdown
* Activity heatmap

### 👤 Profile

* User profile
* Achievement collection
* Class perks
* Language switching
* Settings panel
